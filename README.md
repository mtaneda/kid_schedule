# きょうのよてい (Kid Schedule) 🕒

時計を読み始めたばかりの小さな子どもが、朝の登園のお支度や生活リズムを「じぶんで考えて動く」ためのサポートをするスケジュール管理ウェブアプリケーションです。
家庭内で使うためのものなので、面倒なログイン画面などはありません。

## 🌟 アプリコンセプト

「あと何分で出発？」「今は何するべき？」といった時間の感覚を視覚的に捉えやすくすることで、子どもの自立心を育みます。
本物の **タイムタイマー** のような赤い扇形が徐々に減っていくUIを取り入れており、残り時間を直感的に理解することができます。

## ✨ 主な機能

* **視覚的なタイムタイマー表示**
  本物の **タイムタイマー** と同じように、赤い扇形部分が残り時間に応じて減少し、残り時間が視覚的に把握できます。
* **音声による事前予告と切り替わり**
  予定が切り替わる瞬間だけでなく、「あともう少しで次の予定だよ」という数分前の事前予告音を鳴らすことができます。音声は管理画面から子どもが好きな音（MP3）に変更可能です。
* **おやすみモード**
  指定した夜間の時間は画面を暗くし「おやすみなさい...」のメッセージに切り替わります。
  ※必ずしも液晶の焼き付きを保護できるとは限りません。
* **お天気表示サポート**
  Open-Meteo API を利用し、その日の天気と気温（最高/最低）をアイコンつきで自動表示します。
* **保護者向け管理パネル (`/admin`)**
  一週間のスケジュール登録、事前予告時間の調整、音声ファイルのアップロードができる使いやすい管理画面を備えています。データはローカルのJSONに保存されます。

## 🚀 インストール & 起動方法

### 前提条件

* Python 3.x がインストールされていること

### インストールと起動手順

1. ディレクトリに移動します。
2. 必要なライブラリ（Flask）をインストールします。
   ```bash
   pip install flask
   ```
   必要な場合は venv をご利用ください。
3. アプリケーションを起動します。
   ```bash
   python app.py
   ```
4. ブラウザで以下のURLにアクセスします。
   * **子ども用メイン画面**: `http://localhost:5001/`
   * **保護者用 管理画面**: `http://localhost:5001/admin`

## ⚙️ 管理画面（設定）について

1. `http://localhost:5001/admin` にアクセスします。
2. **「1週間のスケジュール管理」** で、曜日ごとに「始まり」「終わり」「子ども向けのメッセージ」を設定します。（例：「おきるじかんです🌞」）
3. **「基本設定」** で、事前予告を何分前に鳴らすか、おやすみモードの時間を設定します。
4. **「音声ファイルの変更」** から、予告アラーム音や切り替わり音をアップロードできます。

## 💡 使い方

古いタブレット端末（Fire HD 8 など）で表示し、全画面モードで常に画面を点灯させてリビングに置いておく使い方がおすすめです。画面を一度タッチすると全画面になり、音声再生の許可も通るようになっています。
